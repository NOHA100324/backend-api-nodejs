# Frontend Nova Salud

Este frontend es una interfaz estática que consume la API REST del backend.

## Uso

1. Inicia el backend:
   ```bash
   cd backend
   npm install
   npm start
   ```

2. Inicia el frontend:
   ```bash
   cd frontend
   npm install
   npm start
   ```

3. Abre `http://localhost:5500` en el navegador.

El frontend consulta la API en `http://localhost:3000/api/products`.
